# Regressão Linear — Experiência vs. Salário

Modelo de regressão linear simples que prevê salário a partir dos anos de experiência de um profissional.

## Dataset

[Years of Experience and Salary Dataset](https://www.kaggle.com/datasets/rohankayan/years-of-experience-and-salary-dataset) — Kaggle  
30 registros, 2 variáveis: `YearsExperience` e `Salary`.

## Resultados

| Métrica | Valor |
| --- | --- |
| R² | 0.9024 |
| RMSE | $7.059 |
| Coeficiente | $9.423 / ano |
| Intercepto | $25.321 |

## Dependências

```bash
pip install -r requirements.txt
```

Abra `salaryexperience.ipynb` e execute todas as células em ordem.
